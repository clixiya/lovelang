Up & Running in 60 Seconds

Install from npm and run in seconds. If the binary is missing, Lovelang auto-downloads it on first run.

examples/01-romantic-hello.love
~~ romantic greeting
yaad i hai 0
 
dhadkan greet(name) {
  bolo "tum aaye 💌"
  bolo name
  ehsaas sach
}
 
intezaar i chhota hai 3 tab tak
  greet("jaan")
  i hai i milan 1

What It Does

Features Built with Pyaar

Everything you need to write beautiful, emotionally-aware programs.

Desi Syntax
Write code using Hinglish-inspired keywords like yaad, vada, and dhadkan. Reads like a love letter, runs like C.
Human-Mode Phrases
Type natural phrases like baby bolo naa and love you baby byeee — the preprocessor handles normalization before parsing.
CLI with Moods
Run programs in romantic, toxic, or shayari mode. Output tone changes based on the mood flag.
VS Code Extension
Full syntax highlighting, IntelliSense, code snippets, hover docs, and a run button right in the editor. Install the .vsix and you're set.
Word Operators
Use milan (add), doori (subtract), barabar hai (equals). Math has never felt this romantic.
Open Source
Built in pure C with a clean pipeline: preprocess → lex → parse → execute. MIT licensed. Fork it, extend it, make it yours.
Output Modes

Pick Your Vibe

The same program can run in three completely different emotional modes. Your code, your feelings.

🌹
Romantic
--mode romantic
Warm, loving output messages. Program goodbye says "dil se bidaai..." — from the heart.
🔥
Toxic
--mode toxic
Dramatic, passionate output. Every farewell hits like a breakup song. Intense but cathartic.
🎭
Shayari
--mode shayari
Poetic Urdu-inspired tone. Your terminal becomes a mushaira. Code that sounds like verse.
Syntax Showcase

Beauty in Every Line

Lovelang supports both natural desi phrasing and structured core syntax — mix them freely.

Variables & Constants
// Mutable variable
yaad count hai 0
yaad_karo score hai 10
 
// Immutable constant
vada name hai "forever"
 
// Human-mode alias
baby yaad rakho badge hai "gold"
 
// Assignment
count hai count milan 1
Conditionals
// Natural style (preferred)
agar score bada hai 50 toh
  bolo "winner 🏆"
warna agar score bada hai 0 toh
  bolo "thoda aur"
warna
  bolo "next time..."
bas itna hi
 
// One-liner
sun na agar ok toh bolo "yes" warna bolo "no"
Functions
// Define with dhadkan
dhadkan greet(name) {
  bolo "tum aaye: " milan name
  ehsaas sach
}
 
// Call it
yaad ok hai greet("jaan")
 
// Festival block
festival diwali {
  bolo "✨ lights on"
}
Loops & Stop
// intezaar = wait/while
yaad i hai 0
intezaar i chhota hai 3 tab tak
  bolo i
  i hai i milan 1
bas itna hi
 
// Stop execution
love you baby byeee
// also: love_byeee()
VS Code Extension

Love Lang Tools for VS Code

A complete developer experience for writing `.love` files, right inside your editor.

Syntax Highlighting
TextMate grammar with fallback decoration highlighting. Every keyword, string, and operator is beautifully colored.
IntelliSense & Snippets
Autocomplete for all keywords, human-mode phrases, and function patterns. Type dhadkan and let the extension do the rest.
Hover Documentation
Hover over any keyword to get its meaning, aliases, and usage examples inline.
Run from Editor
Run Current File button in the editor title. Supports mode flags, token debug, and custom binary paths via the Settings webview.
Visual Settings Panel
A built-in webview settings panel for configuring run mode, flags, paths, and more — no JSON editing needed.

Installing the Extension

1
Download the VSIX — grab love-lang-tools-1.0.0.vsix from the release assets.
2
Install via CLI:
code --install-extension love-lang-tools-1.0.0.vsix
3
Or install via UI: Extensions panel → ··· menu → Install from VSIX...
4
Open any .love file — language mode auto-switches to Lovelang. Hit the ▶ button to run.
5
Configure: Run Lovelang: Settings from the command palette to set your binary path and default mode.
