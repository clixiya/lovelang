# Love Lang Tools Extension Catalog

## Entry Points

- `package.json`: extension metadata, contributions, commands, settings
- `extension.js`: runtime activation, completions, hovers, run command, settings webview

## Language Support Assets

- `syntaxes/love.tmLanguage.json`: TextMate grammar
- `language-configuration.json`: comment/bracket/autoclose rules
- `snippets/love.json`: user snippets

## Documentation and Licensing

- `README.md`: user-facing extension docs
- `LICENSE`: extension license

## Build Artifacts

- `love-lang-tools-*.vsix`: packaged extension files
